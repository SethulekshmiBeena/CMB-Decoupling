# CMB Photon Decoupling

A Python project simulating hydrogen recombination in the early universe.

## What this computes
- Ionization fraction x_e(z) using the Saha equation
- Coming soon: optical depth τ(z) and visibility function g(z)

## How to run
```bash
pip install numpy matplotlib
python saha.py
```

## Physics
At z ≈ 1100, the universe cooled enough for protons and electrons
to combine into neutral hydrogen. This is when the CMB photons
were released — the "last scattering surface".
